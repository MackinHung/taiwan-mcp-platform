# Contributing

感謝你的興趣！本專案以維護者主導開發，接受針對**現有功能**的 bug 修復與改善。

## 貢獻範圍

- 現有 MCP Server 的 bug 修復、效能改善
- 現有平台功能的問題修正
- 文件修正與翻譯

> 新功能或新 MCP Server 請先開 Issue 討論。

## 提交流程

1. Fork 並建立 feature branch
2. 確保所有測試通過：`npm test`
3. Commit 格式：`<type>: <description>`（feat / fix / docs / test / refactor）
4. 提交 Pull Request

## 本地開發

```bash
git clone https://github.com/MackinHung/taiwan-mcp-platform.git
cd taiwan-mcp-platform
npm install
npm test
```

## 授權

貢獻的程式碼將以 AGPL-3.0 授權釋出。
