常見問題

為什麼平台是免費的？

台灣 MCP 市集是開源專案（AGPL-3.0），致力於讓台灣的公開資料更容易被 AI 助手存取。 平台核心功能永久免費，未來的付費服務（RAG Intelligence）將專注於進階的智慧路由與語義搜索功能。

RAG Intelligence 是什麼？

RAG Intelligence 是平台的進階智慧路由服務。透過語義搜索與向量嵌入技術， 讓 AI 助手能用自然語言找到最適合的 MCP 伺服器與工具， 並根據使用情境智慧推薦相關的資料來源。

RAG Intelligence 何時推出？

RAG Intelligence 目前在開發規劃階段。我們將在準備就緒後公布具體推出時間與定價。 目前所有使用者都可以免費使用平台的完整功能。

可以自架平台嗎？

可以！台灣 MCP 市集採用 AGPL-3.0 開源授權，你可以自由部署在自己的 Cloudflare 帳號上。 詳見 GitHub 專案頁面。
