信任與安全

了解我們如何保護你的資料與平台安全

唯讀代理架構

台灣 MCP 市集採用唯讀代理架構，僅代理台灣政府公開資料， 不儲存、不修改原始資料。你的請求從平台直接轉發至政府 API， 回應原封不動地傳回 — 平台只是一個安全的中間層。

使用者 → MCP 平台（唯讀代理）→ 政府 API（原始資料）

認證安全

OAuth 2.0 登入（GitHub / Google），平台不經手密碼。 PKCE 防護授權碼攔截，CSRF Token 保護每個 session， Secure + HttpOnly Cookie 防止 XSS 竊取。

API 安全

所有通訊 HTTPS 加密。API Key 以 SHA-256 hash 儲存， 即使資料庫被存取也無法還原。Rate Limiting 防止濫用： 匿名 30 次/分鐘，認證 50-2000 次/分鐘（依方案）。

基礎設施

Cloudflare Workers 無伺服器架構，無 SSH 攻擊面。 Cloudflare WAF 網頁應用防火牆 + DDoS 防護自動生效。 基礎設施通過 ISO 27001、SOC 2 Type II 認證。

營養標示

每個 MCP Server 都有透明的安全評分，四大維度標章： 程式碼透明度、資料敏感度、權限範圍、社群信任度。 讓你一目了然每個工具的安全性。

開源已審計 公開資料 唯讀 信賴

異常偵測

平台持續監控 API 使用模式，自動偵測三種異常： API Key 異常使用（暴量請求）、Session 跨國異常、 暴力破解嘗試。發現異常時自動記錄並通知管理員。

法規合規

台灣個人資料保護法（PDPA）合規。 APEC CBPR 跨境隱私保障。 72 小時資料外洩通報承諾。 完整的事件處理 SOP 已就緒。
