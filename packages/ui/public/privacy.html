隱私政策

最後更新：2026 年 3 月 18 日

30 秒摘要

台灣 MCP 市集是一個唯讀代理平台，僅代理台灣政府公開資料。 我們只儲存你的 OAuth 帳號資訊（email、名稱）和 API 使用紀錄。 不收集密碼、不儲存付款資訊、不使用追蹤型 Cookie。 你可以隨時查詢、更正或刪除你的資料。

1. 資料收集範圍

  • 帳號資料：GitHub/Google OAuth 提供的 email、顯示名稱、大頭貼
  • 使用資料：API 呼叫紀錄（endpoint、時間戳、IP、API key hash）
  • 不收集：密碼、付款資訊、生物辨識、位置資訊

2. 資料使用目的

  • 帳號驗證與 session 管理
  • API 用量統計與 rate limiting
  • 平台服務改善（匿名統計，不涉及個人識別）

3. 資料儲存

  • 儲存位置：Cloudflare D1（SQL）、KV（快取）、R2（檔案）
  • 資料中心：Cloudflare 全球 CDN（見「跨境傳輸」章節）
  • 保留期限：帳號資料保留至用戶刪除帳號；API 紀錄保留 90 天

4. 資料分享

  • 不出售、不分享個人資料給第三方
  • 政府機關依法要求時配合提供
  • 匿名統計資料可能用於平台透明度報告

5. 用戶權利（個人資料保護法第 3 條）

你擁有以下權利：

  • 查詢：查詢平台持有的個人資料
  • 更正：要求更正不正確的資料
  • 刪除：要求刪除帳號及相關資料
  • 停止處理：要求停止特定資料處理

6. 跨境傳輸

  • 本平台使用 Cloudflare 全球基礎設施，資料可能經由不同國家的節點傳送
  • 台灣為 APEC CBPR（跨境隱私規則）成員，跨境傳輸受相關規範保障
  • Cloudflare 符合 ISO 27001、SOC 2 Type II 國際安全認證

7. Cookie 政策

  • 僅使用 session cookie（OAuth 驗證用途）
  • 不使用追蹤型 cookie 或第三方 cookie
  • Cookie 存在時間與 session 相同（最長 24 小時）

8. 修改通知

  • 隱私政策修改時於平台公告
  • 重大變更以 email 通知已註冊用戶
  • 繼續使用平台即視為同意最新版本之隱私政策
