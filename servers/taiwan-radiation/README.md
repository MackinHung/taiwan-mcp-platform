# @formosa-mcp/taiwan-radiation

Taiwan MCP server for AEC (Atomic Energy Council) radiation monitoring data.

## Tools

| Tool | Description |
|------|-------------|
| `get_current_radiation` | Get current radiation readings across Taiwan |
| `search_by_region` | Search radiation stations by county/district |
| `get_radiation_alerts` | Get radiation alert information |
| `get_station_history` | Get history data for a specific station |
| `get_radiation_summary` | Get radiation monitoring summary statistics |

## Usage

```bash
npx @formosa-mcp/taiwan-radiation
```
