# @formosa-mcp/taiwan-movie

Taiwan MCP server for movie and cinema data from the Ministry of Culture (MOC) open data platform.

## Tools

| Tool | Description |
|------|-------------|
| `search_movies` | Search movies/film events by title keyword |
| `search_cinemas` | Search cinema venues/screening locations |
| `get_showtimes` | Get showtimes for a specific movie/event |
| `get_movie_details` | Get full movie/event details |
| `get_new_releases` | Get latest/upcoming movie events |

## Data Source

- **cloud.culture.tw** — MOC open data (category 8: cinema events)

## Usage

```bash
npx @formosa-mcp/taiwan-movie
```
